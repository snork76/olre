Old Lyme Reassessment Maps

Notes: The maps below are based on the public data from VGSI for Old Lyme, CT.
The data was collected around April 22, 2025, and may not cover every property in the town. I also can't vouch for the accuracy of any of the data in the database, or presented in the maps. Use it at your own risk ;-).

Clicking on the titles below will bring up an interactive map that you can zoom in and out of. Each of the properties will have a pop-up with more details if you click on a dot.

A few statistics about the data:

  • The average appraisal value in 2024 is $641,281 (median: $546,000)
  • The average building is 1776sqft (median: 1478sqft)
  • The average lot size is 2.33ac (median: 0.56ac)
  • The average appraisal value increased by 60.6% between 2023 and 2024 (median: 69.3%)
  • The average land appraisal increased by 49% between 2023 and 2024 (median: 43.5%)
  • The average improvements appraisal increased by 72.6% between 2023 and 2024 (median: 85.3%)

  • Sales By Year Map

    • This map plots a dot for a house sold in Old Lyme between 2020 and 2025. The color represents the year the property was sold. I've excluded $0 and $1 sales, and I only looked at the most recent sale. So if a house sold in 2021 for $300,000 and then the property was transferred to a trust for $0 in 2022, it would be excluded from this map.
  • Lot Size Map

    • This map breaks the lot sizes up into 10 buckets and gives each a color. The buckets are:
    0ac, 0.1ac, 0.25ac, 0.5ac, 0.75ac, 1.0ac, 2.0ac, 4.0ac, 5.0ac, >5.0ac
  • Total Value Increase Map
    • This map plots the increase in the total appraisal value (land + improvements) from 2023 to 2024. the data is grouped into buckets like: decrease - 0%, 1-10%, 11-20% ... 90-100%, >100%.
  • Neighborhood Map
    • This is a plot of the various "neighborhoods" in Old Lyme. Each property record has a Neighborhood code associated with it. Each code has a different color on the map. This becomes interesting when you compare it with the "Land Value Increase Map" below. There appears to be a correlation between the neighborhood and the increase in land value.
  • Land Value Increase Map
    • This map shows the percentage increase (or decrease) in the land value of a property from 2023 to 2024. This data is not bucketed, but rather it's the colors are normalized from -100% (red) - 100% (green).
  • Price Per Acre Map
    • This map plots the price per acre of land. The data is bucketed into groups of $100,000 per acre, and then a catch-all for land above $1,000,000 per acre. Properties close to the water clearly stand out.
  • Improvements Value Increase Map
    • The improvements value map shows the percentage increase (or decrease) in the improvements (the house) on a property between 2023 and 2024. This data is bucketed into groups of 10% with a decrease-0% bucket and a >100% bucket on either end.
  • Appraisal Map
    • This map shows the appraisal value of the properties. The data is normalized between yellow (the lowest priced property) and dark red (the highest priced property). This map only includes records where the Description contains "Single" or "SFR" (Single Family Residence). When it included the schools at multiple millions of dollars, the data got skewed.

Filtered Data

The maps below look at properties that are:

  • 2,200 to 4,000 square feet
  • 0.5 to 1.5 acres
  • less than $1.25M

  • Filtered Appraisal Map
    • This map shows the appraisal value of the properties that match the filter above. The data is normalized between yellow (the lowest priced property) and dark red (the highest priced property).
  • Filtered Total Value Increase
    • This map shows the increase in the total appraisal value (land + improvements) from 2023 to 2024 for the properties that match the filter above. The data is grouped into buckets like: decrease - 0%, 1-10%, 11-20% ... 90-100%, >100%.
